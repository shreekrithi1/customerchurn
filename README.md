# Customer Churn Prediction (ML)

Predicts telecom customer churn via **supervised binary classification**.

## Dataset
- `WA_Fn-UseC_-Telco-Customer-Churn.csv` (7,043 rows, 21 features)
- Target: `Churn` (Yes/No)
